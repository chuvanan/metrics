*** Installation

Install the stable version of =metrics= from CRAN:

#+BEGIN_SRC r
install.packages("metrics")
#+END_SRC

Or install the development version from Github with:

#+BEGIN_SRC r
devtools::install_github("chuvanan/metrics")
#+END_SRC
